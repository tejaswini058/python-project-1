# 🐍 Snake Water Gun Game

A fun command-line game built in Python where you battle the computer 
in a classic Snake, Water, Gun showdown!

## What it does
- Play unlimited rounds against the computer
- Computer picks randomly each round
- Tracks your score, computer score, and ties live
- Shows a final score summary when you quit

## Technologies Used
- Python
- Random module (built-in)

## How to Run
1. Make sure Python is installed on your system
2. Download or clone this repository
3. Run the following command:
   python snake_water_gun.py

## Game Rules
- Snake beats Water 🐍💧
- Water beats Gun 💧🔫
- Gun beats Snake 🔫🐍
